# Doctor-Strange-CV

A real-time "Doctor Strange"-style magical effects system using Python, OpenCV, and MediaPipe Hands.

## Features
- **Magic Circles**: Hold up an open palm to summon glowing rotating magic circles.
- **Portals**: Hold up both hands and move them apart to open a glowing portal.
- **Spark Trails**: Particles emit from your fingertips as you move your hands.

## Setup
1. Clone the repository.
2. Ensure you have Python installed.
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `python main.py`
